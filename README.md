# 🤖 JARVIS AI Assistant

Advanced AI assistant with voice commands and OpenAI integration.

## Quick Start

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Set OpenAI API key in `.env`:
```
OPENAI_API_KEY=your_key_here
```

3. Run JARVIS:
```bash
python run_jarvis.py
```

## Commands

- "open notepad" - Opens applications
- "what time is it" - System information  
- "search for cats on google" - Web searches
- "hey bro open calc now" - Casual commands with typos

## Files

- `run_jarvis.py` - Main launcher
- `simple_jarvis.py` - Basic version
- `advanced_jarvis.py` - Full featured version
- `jarvis_gui.py` - GUI interface
